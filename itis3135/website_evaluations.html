Good and Bad Web Websites

Website Evaluations

Good Website Name

Apple

Description of the good website and why it's well-designed.

  • Contrast: A good website uses contrast to make important stuff stand out. This can be through colors, fonts, or spacing—like making text pop against the background so it’s easy to read.
  • Repetition: Keeping things consistent, like using the same fonts, colors, and styles across the site, makes it look more put-together and professional. It also helps with branding.
  • Alignment: Everything should line up nicely so the site looks neat and organized. When things are properly aligned, it’s easier to read and navigate.
  • Proximity: Stuff that belongs together should be close together. Grouping related content makes it easier for users to find what they need without feeling lost.

Bad Website Name

J. C. Penny

Description of the bad website and why it's poorly designed.

  • Contrast: The text and background don’t have enough contrast, making it hard to read in some areas, especially in product categories. Some text gets lost on the background, which can be annoying for users.
  • Repetition: There’s too much repetition of similar elements like banners, buttons, and promotional deals all over the site. It makes it feel cluttered and repetitive without any real focus.
  • Alignment: The layout is a bit all over the place. Some content is misaligned, and things don’t line up properly. This makes the site feel unorganized and hard to navigate.
  • Proximity: Information isn’t grouped properly. You’ve got products and promotions all jumbled together without enough spacing between them, making it tough to know where to focus.
