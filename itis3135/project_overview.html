Page Design

Home
Purpose: Welcomes incoming users and viewers to the site, an overall guide.
Audience: Comic Readers
Content: Loud Comic behind the scenes content, Content news and fanart page feed, contact info, character info.
Actions: Button counter, moving objects with hover, highlighted text with hover, reply button, links.

About
Purpose: Give insight on the how the Comic was created, what purpose, the creators motivations and goals for this project.
Audience: Clients, collaborators, Artists.
Content: Biography, values, funding goal.
Actions: navigation.

Character Info
Purpose: Display projects visually.
Audience: Fans, Writers, Comic Readers
Content: Images, descriptions, tags.
Interactivity: Image viewer, category filter.

Contact
Purpose: Show client feedback.
Audience: clients.
Content: creator credentials.
Interactivity: Navigation.

Loud Feed
Purpose: Allow user to reach out, share thoughts and feelings, be apart of the Loud community.
Audience: Anyone interested in the series.
Content: Social Media feed
Interactivity: JavaScript enabled.
Validation: Required fields
Actions: post button, reply button, like and dislike button, image post feature.

Dynamic Functionality

Loud Feed: Inteactive social media page. reply, comment buttons, posts, post interactive buttons

Home Page: moving text blocks.
